# Wiki Links 사용법

Wiki Links는 노트 간의 연결을 만드는 핵심 기능입니다.

## 기본 문법

노트를 연결하려면 이중 대괄호를 사용하세요:

```markdown
[[노트 제목]]
```

예시: [[Welcome to Synapse]]

## 표시 텍스트 변경

다른 텍스트로 표시하고 싶을 때:

```markdown
[[노트 제목|표시할 텍스트]]
```

예시: [[Getting Started|시작하기 가이드]]

## 존재하지 않는 노트 링크

아직 없는 노트에 링크할 수 있습니다:

```markdown
[[나중에 만들 노트]]
```

링크를 클릭하면 해당 노트를 생성할 수 있습니다.

## 그래프에서 보기

모든 Wiki Links는 그래프 뷰에서 시각화됩니다:
- 노드: 각각의 노트
- 엣지: 노트 간의 연결

## 백링크

노트 하단에서 현재 노트를 참조하는 다른 노트들을 볼 수 있습니다.
