# Welcome to Synapse

Synapse is a local-first markdown note-taking app with powerful linking and visualization features.

## Features

- **Markdown Editor**: Write notes in markdown with live preview
- **Wiki Links**: Connect your notes with [[wiki-links]]
- **Graph View**: Visualize connections between notes
- **Local-First**: All your data stays on your machine
- **Publish**: Share your notes as a static website

## Getting Started

1. Create a new note by clicking the "New Note" button
2. Use `[[Note Name]]` to link to other notes
3. View the graph to see connections
4. Publish your notes when ready
